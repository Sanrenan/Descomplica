java nomedoarquivo.java - executa o arquivo java
javac nomedoarquivo.java - compila o .java e cria um arquivo .class
path C:\Program Files\Java\jdk-18.0.1.1\bin - indica o caminho do compilado e executado do java, para que rode no cmd.

cd - para ir até o diretório onde estão os arquivos java.

dir *.java - verifica se os arquivos java estao no diretorio q você configurou o path.
